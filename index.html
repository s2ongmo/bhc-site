01 About // who we are

BHC는 실전 취약점 연구에 집중하는 보안 리서치 팀입니다. 버그바운티 플랫폼과 직접 제보 채널을 통해 실제 운영 중인 서비스와 소프트웨어의 결함을 찾아내고, 벤더가 수정할 수 있도록 검증된 PoC와 함께 제보합니다.

단발성 제보가 아니라 재현 가능하고 신뢰할 수 있는 리서치를 지향합니다. 모든 발견은 책임 있는 공개 원칙을 따르며, 패치 이전에는 세부 내용을 공개하지 않습니다.

웹·모바일 같은 애플리케이션 계층부터 Windows / Linux 커널, 데스크톱 클라이언트, 오픈소스 코드 오딧까지 — 공격 표면을 가리지 않습니다.

02 AI-Driven Research // hunting at machine scale

세상은 AI에 맞게 바뀌고 있고, 취약점 헌팅도 마찬가지입니다. BHC는 LLM 기반 분석 파이프라인으로 정찰·코드 오딧·변형(variant) 탐색을 자동화해 사람이 놓치기 쉬운 공격 표면을 더 빠르게, 더 넓게 훑습니다.

→ ai.01

AI Red Teaming

LLM·에이전트 시스템 자체를 공격 대상으로. 프롬프트 인젝션, 도구 오남용, 권한 경계 우회까지 — AI 제품의 보안을 AI로 검증합니다.

→ ai.02

$10,000+ with AI

AI 보조 헌팅 워크플로우로 실제 바운티를 달성했습니다. 인증 프록시 패턴, operation_id 리플레이 등으로 취약점 발굴을 스케일합니다.

→ ai.03

Automated Audit

대량 바이너리·소스 코드를 LLM 파이프라인에 태워 CVE 변형을 탐색. 헤드리스 디컴파일과 결합해 사람의 직관을 기계 규모로 확장합니다.
